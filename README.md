# How to learn

Это мой первый учебный проект. Одностраничный сайт с информацией о том, как нужно учиться._
**Сайт рассказывает о** проблемах в обучении, техниках обучения, принципах обучения,
содержит два видео с TED, ссылку на книгу по обучению и пр.

## Используются технологии:

* Разметка HTML,
* Стили CSS,
* Флексбоксы.
* Анимации в CSS,
* Фреймы,
* Разбивка блоков согласно Nosted БЭМ.
